# Ore Slice Classifier

## Тема проекта - "Скажи мне кто твой шлиф"

**Мультимодальный сервис анализа материаловедческих данных с авто-контролем качества**

Проект разрабатывается для автоматического анализа полированных шлифов руды и материаловедческих данных с использованием методов машинного обучения и компьютерного зрения.

Система обрабатывает микроскопические изображения шлифов, сегментирует минеральные фазы, оценивает содержание талька, классифицирует сульфидные включения и формирует интерпретируемый результат для эксперта.

## Основная идея

Мы строим модульный ML/CV-пайплайн для анализа геологических и материаловедческих данных.

Основные задачи системы:

* сегментация сульфидных включений на OM/SEM-изображениях;
* классификация срастаний на обычные и тонкие;
* детекция и количественная оценка доли талька;
* расчёт площадей и процентных долей фаз;
* построение цветовой маски поверх исходного изображения;
* формирование итоговой классификации руды;
* отображение метрик и результатов в Streamlit-интерфейсе.

## Классы результата

Система классифицирует образец по экспертной логике:

1. Если доля талька больше 10% — **оталькованная руда**.
2. Если талька не больше 10% и преобладают обычные срастания — **рядовая руда**.
3. Если талька не больше 10% и преобладают тонкие срастания — **труднообогатимая руда**.

## Установка проекта

Клонировать репозиторий:

```bash
git clone https://github.com/YOUR_USERNAME/ore-slice-classifier.git
cd ore-slice-classifier
```

Создать виртуальное окружение:

```bash
python -m venv .venv
```

Активировать окружение на Windows:

```bash
.venv\Scripts\activate
```

Установить зависимости:

```bash
pip install -r requirements.txt
```

## Запуск Streamlit-приложения

```bash
streamlit run app/streamlit_app.py
```

После запуска приложение откроется в браузере.
